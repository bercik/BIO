BIO Documentation

Basic informations

data types

array - series of elements of any type. Mutable

bool - has only two posible values: true and false. Immutable

dict - collections of pairs <string, any>. Mutable

error - contains informations about error that appears. Immutable

float - single-precision 32-bit IEEE 754 floating point. Immutable

int - 32-bit signed two's complement integer, which has a minimum value of -231 and a maximum value of 231-1. Immutable

none - represents absent of value. Immutable

string - represents sequence of characters. Immutable

tuple - series of elements of any type. Immutable

errors

Errors are values of type error which are returned from function where an error occur. If you don't handle an error, that is don't assign it to variable or pass to another function, then onUNHANDLED_ERROR event will occur (see in basic). It will occur also when there is error returned from onSTART function. There are two cases when error can cause function to return. Both are when you pass not bool to FOR or IF function as condition. Notice that there are few functions that takes error as parameter. Other will return error with cause as this given error parameter. This behavior is to prevent errors from disappearing
