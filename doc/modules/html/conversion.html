conversion

TO_STRING (val)

Converts given val to string

Parameters:

val (all)

Return value:

(string)

TO_INT (val)

Converts given val to int. In case of bool convert true to 1 and false to 0. In case of float rounds number to the nearest int. In case of string tries to convert it to int

Parameters:

val (bool, float, int, string)

Return value:

(int)

Errors:

BAD_PARAMETER_TYPE - If val is not bool, float, int or string

CONVERSION_ERROR - If val is string but cannot be converted to int

TO_FLOAT (val)

Converts given val to float. In case of bool convert true to 1.0 and false to 0.0. In case of string tries to convert it to float

Parameters:

val (bool, float, int, string)

Return value:

(float)

Errors:

BAD_PARAMETER_TYPE - If val is not bool, float, int or string

CONVERSION_ERROR - If val is string but cannot be converted to float

TO_BOOL (val)

Converts given val to bool. In case of int or float convert zero value (0 or 0.0) to false and any non zero value to true. In case of string converts "true" to true and "false" to false

Parameters:

val (bool, float, int, string)

Return value:

(bool)

Errors:

BAD_PARAMETER_TYPE - If val is not bool, float, int or string

CONVERSION_ERROR - If val is string but cannot be converted to bool

TO_ARRAY (col)

Converts given col to array. If you pass array it would return it. If you pass tuple it would copy it and return as array

Parameters:

col (array, tuple)

Return value:

(array)

Errors:

BAD_PARAMETER_TYPE - If col is not array or tuple

TO_TUPLE (col)

Converts given col to tuple. If you pass tuple it would return it. If you pass array it would copy it and return as tuple

Parameters:

col (array, tuple)

Return value:

(tuple)

Errors:

BAD_PARAMETER_TYPE - If col is not array or tuple
