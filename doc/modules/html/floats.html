CEIL (val)

Gives the smallest int that is greater than or equal to the passed val

Parameters:

val (int, float)

Return value:

(int)

Errors:

BAD_PARAMETER_TYPE - If val is not int or float

FLOOR (val)

Gives the largest integer that is less than or equals to the passed val

Parameters:

val (int, float)

Return value:

(int)

Errors:

BAD_PARAMETER_TYPE - If val is not int or float
