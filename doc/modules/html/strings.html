SUBSTR (str, start, end)

Returns a new string that is a substring of str in range <start, end)

Parameters:

str (string)

start (int)

end (int)

Return value:

(string)

Errors:

BAD_PARAMETER_TYPE - If str is not string or start or end is not int

START_GREATER_THAN_END - If start is greater than end

INDEX_OUT_OF_BOUNDS - If start is less than zero or end is greater or equals str length

TO_LOWERCASE (a)

Converts all of the characters in string a to lower case and returns as new string. If you pass collection than it would convert all strings in it to lower case and returns as new array

Parameters:

a (string, array, tuple)

Return value:

(string, array) - if you pass string returns string, if you pass array or tuple returns array

Errors:

BAD_PARAMETER_TYPE - If a is not string, array or tuple or collection contains not only strings

REPLACE (str, b, replacement)

If b is string than replaces each substring of str that matches the given regular expression with the given replacement. Because this function internally uses java String.replaceAll function, check java doc for informations about creating regular expressions. If b is int than replaces characters in str starting from position b by replacement string. If necessary resulting string can be longer than original str

Parameters:

str (string) - original string

b (string, int) - regular expression or index

replacement (string) - replacement string

Return value:

(string) - new string

Errors:

BAD_PARAMETER_TYPE - If str or replacement is not string or b is not int or string

INDEX_OUT_OF_BOUNDS - If b is int and it is less than zero or greater than str length

REGEX_ERROR - If b is string and it is invalid regular expression
