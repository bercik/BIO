SUBSTR (str, start, end)

Returns a new string that is a substring of str in range <start, end)

Parameters:

str (string)

start (int)

end (int)

Return value:

(string)

Errors:

BAD_PARAMETER_TYPE - If str is not string or start or end is not int

START_GREATER_THAN_END - If start is greater than end

INDEX_OUT_OF_BOUNDS - If start is less than zero or end is greater or equals str length

TO_LOWERCASE (str)

Converts all of the characters in str to lower case and returns as new string. If you pass collection than it would convert all strings in it to lower case and returns as new array

Parameters:

str (string, array, tuple)

Return value:

(string, array) - if you pass string returns string, if you pass array or tuple returns array

Errors:

BAD_PARAMETER_TYPE - If str is not string, array or tuple or collection contains not only strings
