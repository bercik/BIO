RSHIFT (num, shift)

Right shifts num by shift bits. If you pass collections than element at every index in first collection would be right shifted by element at same index in second collection bits

Parameters:

num (int, array, tuple)

shift (int, array, tuple)

Return value:

(int, array) - if you pass ints right shifted int. If you pass collections array of right shifted ints

Errors:

BAD_PARAMETER_TYPE - If num or shift is not int, array or tuple or if you mixed ints and collections or if collections contains not only ints

LSHIFT (num, shift)

Left shifts num by shift bits. If you pass collections than element at every index in first collection would be left shifted by element at same index in second collection bits

Parameters:

num (int, array, tuple)

shift (int, array, tuple)

Return value:

(int, array) - if you pass ints left shifted int. If you pass collections array of left shifted ints

Errors:

BAD_PARAMETER_TYPE - If num or shift is not int, array or tuple or if you mixed ints and collections or if collections contains not only ints

XOR (a, b)

Perform xor (exclusive or) operation on two given ints. If you pass collections then it will xor every element from both collections at given index

Parameters:

a (int, array, tuple)

b (int, array, tuple)

Return value:

(int, array) - if you pass ints xored int. If you pass collections array of xored ints

Errors:

BAD_PARAMETER_TYPE - If a or b is not int, array or tuple or if you mixed ints and collections or if collections contains not only ints
