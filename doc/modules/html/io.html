PRINT (<val>*)

Prints values val on standard output (typically screen) separated by space

Parameters:

val (all) - value to print

Return value:

(string) - printed string

PRINTLN (<val>*)

Prints values val on standard output (typically screen) separated by newlines

Parameters:

val (all) - value to print

Return value:

(string) - printed string

INPUT (<prompt>*)

If the prompt argument is present, it is written to standard output (typically screen) without a trailing newline. The function then reads a line from standard input (typically console), converts it to a string (stripping a trailing newline), and returns that. If there are repeated prompt arguments than there will be showed one after each other and for each one string from input would be read. The whole would be returned as array.

Parameters:

prompt (string) - string to be showed to user

Return value:

(string, array) - If you pass zero or one prompt argument returns string, otherwise array of strings

Errors:

BAD_PARAMETER_TYPE - If any prompt argument is not string
