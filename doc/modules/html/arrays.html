arrays

CREATE_TUPLE (<el>)

Creates tuple of given elements el

Parameters:

el (all) - element to be inserted in tuple

Return value:

(tuple) - tuple of given elements

GET (col, index)

Gets element at given index from array or tuple

Parameters:

col (array, tuple) - collection from which we want to get element

index (int) - which element do we want to get

Return value:

(all) - returns element from col at given index

Errors:

BAD_PARAMETER_TYPE - if passed argument is not array or tuple

INDEX_OUT_OF_BOUNDS - if given index is greater or equal collection size
