Data types

array - series of elements of any type. Mutable

bool - has only two posible values: true and false. Immutable

dict - collections of pairs <string, any>. Mutable

error - contains informations about error that appears. Immutable

float - single-precision 32-bit IEEE 754 floating point. Immutable

int - 32-bit signed two's complement integer, which has a minimum value of -231 and a maximum value of 231-1. Immutable

none - represents absent of value. Immutable

string - represents sequence of characters. Immutable

tuple - series of elements of any type. Immutable

observer

ATTACH_TO_EVENT (event, callback, <event, callback>)

Attach callback functions to event functions. When event function is called, then callback functions attached to it are executed in the same order that they were added and at the end event function is executed. If there are attached callbacks to this callbacks there are also executed in the same manner. Values returned form callback functions are ommited. If error occurs during execution of this function then no callback would be attached to event

Parameters:

event (id) - function to be observed

callback (id) - function to be attached

Return value:

(none)

Errors:

CALLBACK_ALREADY_ATTACHED - if callback is already attached to event

IS_ATTACHED (event, callback)

Checks whether callback is attached to event

Parameters:

event (id)

callback (id)

Return value:

(bool) - true if callback is attached to event, otherwise false

DETACH_FROM_EVENT (event, callback, <event, callback>)

Detach callback functions from event functions. If error occurs during execution of this function then no callback would be detached from event

Parameters:

event (id) - observed function

callback (id) - function to be detached

Return value:

(none)

Errors:

CALLBACK_NOT_ATTACHED - if callback is not attached to event
