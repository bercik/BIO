ASSIGN_LOCAL (id, val, <id, val>)

Creates or updates local variables with identifier id and values val

Parameters:

id (id) - variable identifier

val (all) - variable value

Return value:

(all) - last given value

ASSIGN_GLOBAL (id, val, <id, val>)

Creates or updates global variables with identifier id and values val

Parameters:

id (id) - variable identifier

val (all) - variable value

Return value:

(all) - last given value

IS_LOCAL (id)

Checks whether variable with identifier id exists and is local

Parameters:

id (id) - variable identifier

Return value:

(bool) - true if variable exists, otherwise false

IS_GLOBAL (id)

Checks whether variable with identifier id exists and is global

Parameters:

id (id) - variable identifier

Return value:

(bool) - true if variable exists, otherwise false

GET_GLOBAL (id)

Returnes global variable with identifier id

Parameters:

id (id) - global variable identifier

Return value:

(all) - global variable value

Errors:

NO_GLOBAL_VARIABLE - if there is no global variable with id identifier

RETURN (val)

Returnes value val from function

Parameters:

val (all) - value to be returned

onUNHANDLED_ERROR (err)

Event called when there is error value returned from onSTART function

Parameters:

err (error) - error returned from onSTART function
